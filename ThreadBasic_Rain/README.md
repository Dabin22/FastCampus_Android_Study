ThreadBasic_Rain
=================


#동작 방식
 >1. start버튼을 누르면 파란색 원(빗방울)이 랜던한 크기와 위치, 속도로 떨어집니다.
 >2. stop버튼을 누르면 멈춥니다. 다시 start버튼을 누르면 멈춰있던 원이 다시 떨어집니다.
 
#공부한 내용
 빗방울이 떨어지는 효과를 위해서 Thread가 3개 필요 하였다. 빗방울에 대한 크기,속도,위치값을 가진 쓰레드
 이 쓰레드를 커스텀 뷰에 리스트로 만들어서 거기에 넣는 방식으로 빗방울을 추가해 나갔다. 지정한 위치까지 도착하면 리스트에서
 지워지는 방식으로 구현 그리고 stop 버튼에서 멈추었다가 다시 start버튼을 누르면 동작하기 위해 쓰레드를 하나(CallDraw Thread)
 사용하였다.(여기서 start버튼 사용시 무한으로 빗방울 쓰레드를 만들었다.) 그리고 0.05초마다 빗방울 전체가 움직이는 모습을 보여주기 
위해 postInvalidate()를 해주는 쓰레드하나(callDraw)를 사용해 만들었습니다.
